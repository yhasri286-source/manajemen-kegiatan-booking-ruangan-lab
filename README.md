## PENDAHULUAN

Aplikasi Manajemen Kegiatan dan Booking Ruangan Lab merupakan sebuah sistem berbasis web yang dibuat untuk membantu proses pengelolaan serta peminjaman ruangan laboratorium. Sistem ini dirancang agar proses pemesanan ruangan dapat dilakukan secara teratur, efisien, dan terdokumentasi dengan baik.

Aplikasi ini dibuat sebagai bagian dari tugas pada mata kuliah Pengembangan Web di Universitas Mega Buana Palopo, dengan tujuan untuk menerapkan konsep pemrograman web, pengelolaan basis data, serta perancangan sistem informasi sederhana.

---

## TUJUAN PEMBUATAN SISTEM

Tujuan dari pembuatan aplikasi ini adalah:

1. Mempermudah pengelolaan data ruangan laboratorium
2. Memudahkan pengguna dalam melakukan pemesanan ruangan
3. Membantu admin dalam mengelola dan memantau penggunaan ruangan
4. Menyediakan laporan kegiatan penggunaan laboratorium
5. Menerapkan teori pemrograman web ke dalam bentuk aplikasi nyata

---

## DESKRIPSI SISTEM

Sistem Manajemen Kegiatan dan Booking Ruangan Lab adalah aplikasi berbasis website yang memungkinkan pengguna melakukan pemesanan ruangan laboratorium sesuai dengan jadwal yang dibutuhkan. Admin memiliki hak akses untuk mengelola data pengguna, data ruangan, serta melihat laporan kegiatan.

Sistem ini dikembangkan menggunakan bahasa pemrograman PHP dengan database MySQL, serta antarmuka berbasis HTML, CSS, dan JavaScript.

---

## FITUR SISTEM

1. Login dan Logout
   Pengguna harus melakukan login terlebih dahulu untuk dapat mengakses sistem. Fitur logout digunakan untuk keluar dari sistem dengan aman.

2. Dashboard
   Halaman dashboard menampilkan menu utama dan informasi dasar terkait sistem.

3. Manajemen Data User
   Admin dapat menambah, mengubah, dan menghapus data pengguna yang terdaftar.

4. Manajemen Data Ruangan
   Admin dapat mengelola data ruangan laboratorium yang tersedia, seperti menambah ruangan baru, mengedit data ruangan, dan menghapus data ruangan.

5. Booking Ruangan
   Pengguna dapat melakukan pemesanan ruangan laboratorium berdasarkan tanggal dan waktu yang ditentukan.

6. Laporan Kegiatan
   Sistem menyediakan laporan penggunaan ruangan laboratorium sebagai dokumentasi kegiatan.

## STRUKTUR APLIKASI

Aplikasi ini terdiri dari beberapa bagian utama, yaitu:

* Folder assets untuk menyimpan file pendukung tampilan
* Folder config untuk pengaturan koneksi database
* Folder includes untuk file pendukung sistem
* Folder models untuk pengolahan data dan database
* Folder views untuk tampilan halaman
* File PHP utama sebagai fungsi inti aplikasi

---

## TEKNOLOGI YANG DIGUNAKAN

Teknologi yang digunakan dalam pengembangan aplikasi ini meliputi:

* PHP sebagai bahasa pemrograman backend
* MySQL sebagai sistem manajemen basis data
* HTML, CSS, dan JavaScript untuk antarmuka pengguna
* Web server Apache (XAMPP atau sejenisnya)

---

## CARA MENJALANKAN APLIKASI

Langkah-langkah menjalankan aplikasi ini adalah:

1. Menginstal web server seperti XAMPP atau Laragon
2. Menyalin folder project ke dalam direktori server
3. Mengimpor database dari folder sql ke phpMyAdmin
4. Menyesuaikan konfigurasi database pada folder config
5. Mengakses aplikasi melalui browser menggunakan alamat localhost
